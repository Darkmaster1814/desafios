Cómo navegar?
Index es una one page con un header vinculado para cada sección.
Dentro de cada sección hay un vinculo para ver mas detalles, al hacer click, dentro el header es capaz de mostrar diferentes opciones dentro de las pages y también despliega opciones adicionales.

Usando la sección personal despliega un apartado que se pretende sea una acceso para iniciar seción dentro del sistema
El sistema consta de una sección de articulos o comandas para registrar cada uno de los pedidos en ese momento, para visualizar cada opción dar click en el icono en forma de carrito.
Para observar los pedidos pendientes se debe clickear en comandas, dentro de ella se ven las comandas ingresadas y datos relevantes.
Al hacer click despliega información relevante del pedido y funciona como un gestor previo para mantener un estatus de inventarios y de ventas del comercio diario
Al cerrar sesión vuelve al password y deja accesar a las otras opciones.

Por el momento algunas opciones de esta parte están desabilidadas, la intención es darles funcionalidad posteriormente con ayuda de leguajes de programación adecuados JAVA SQL
